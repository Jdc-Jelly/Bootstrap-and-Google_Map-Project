Capability 1

is_wireless_device

The is_wireless_device comes with three possible values: true, false, or NULL. If the users device is a laptop, whether it has a wireless network card or not, the output value will be false.

Capability 2

built_in_camera

The built_in_camera check can be used to see if the extistance of a camera is present allowing the user to take a picture.

Capability 3

device_class

Using device_class gives you an abstract collection of common characteristics that define a group of devices and their browsers. Ex: higher_mobile, simpler-mobile, is_tablet.
